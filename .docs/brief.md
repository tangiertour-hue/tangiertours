# TangierTours
- One-line positioning: Premium SEO-first tourism website for a licensed Tangier-based tour guide, designed to rank on Google and convert visitors into paying customers.
- Target users: International travelers (US, UK, EU, Israel) seeking private tours, heritage tours, shore excursions, and luxury Morocco experiences.
- Core features:
  1. SEO landing pages for 11 service/destination pages (Tangier Private Tours, Jewish Heritage, Shore Excursions, Casablanca Jewish Heritage, Chefchaouen Day Trips, Private Transfers, Luxury Tours, Camel Ride, About, Reviews, Contact)
  2. Schema.org structured data (LocalBusiness, TouristAttraction, Tour, FAQ, BreadcrumbList)
  3. XML sitemap + robots.txt + canonical URLs
  4. Booking request form + WhatsApp CTA button
  5. Breadcrumb navigation on all pages
  6. Optimized metadata, H1/H2/H3 hierarchy, internal linking
- Important features (P1):
  1. i18n-ready architecture (EN primary; FR, ES, AR, HE later)
  2. Blog section structure (categories: Travel Tips, Heritage, Morocco Culture, Itineraries)
  3. Customer reviews page with testimonial schema
  4. Mobile-first responsive design
- Device strategy: adaptive
- Design style: elegant, luxurious, trustworthy — deep navy + gold palette, serif/sans-serif pairing
- Technical constraints: Next.js App Router, Tailwind CSS, no native Node addons
- Nova Agent: not needed
- Completed: Phase 1 – Requirements captured
- Current iteration: Phase 1 – Site Architecture Proposal
