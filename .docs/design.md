# TangierTours Design System

- Primary color: Deep Navy #0F1F3D (trust, premium)
- Accent color: Warm Gold #C9A84C (luxury, Morocco)
- Background: Warm Ivory #FAF8F5 (elegant, not stark white)
- Text: Charcoal #1A1A2E
- Muted text: #6B7280
- Border: #E8E0D0

- Display font: Playfair Display (serif — elegance, authority)
- Body font: Source Sans 3 (readable, clean, modern)
- UI font: Source Sans 3 medium

- Border radius: 4px (refined, not rounded/playful)
- Hero: full-viewport with dark overlay on photo, gold headline accent
- Cards: subtle shadow, ivory background, gold top-border accent
- Buttons: Primary = navy fill + gold border; Secondary = outlined navy
- Spacing: generous — luxury brands breathe
